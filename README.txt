для создания репника git init в корне репозитория
для добавление файлов в коммит (если добавляем все) git add*
git commit -m "message" для коммита
для привязки удаленного репозитория git remote add origin "path to repo"
для отправки на сервер git push
для запуска аллюра allure serve "путь к репортам"
для запуска тестов npx codeceptjs run --plugins allure
